# just-gdal

Build gdal on macOS. 

## Install

```bash
just download
just build
```

## Usage

```bash
export GEOS_LIBRARY_PATH=/Users/alex.clark/Developer/just-gdal/local/gdal-3.8.5/lib/libgeos_c.dylib
export GDAL_LIBRARY_PATH=/Users/alex.clark/Developer/just-gdal/local/gdal-3.8.5/lib/libgdal.dylib
```
