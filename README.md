# just-gdal

> [!NOTE]  
> Works with Python 3.12

Build gdal on macOS. 

## Install

```bash
just download
just build
```

## Usage

```bash
export GEOS_LIBRARY_PATH=~/Developer/just-gdal/local/gdal-3.8.5/lib/libgeos_c.dylib
export GDAL_LIBRARY_PATH=~/Developer/just-gdal/local/gdal-3.8.5/lib/libgdal.dylib
```
